# 📋 CRM Dashboard

This is a full-stack CRM (Customer Relationship Management) dashboard built using the MERN stack. The project helps manage customer information through a clean, responsive interface while providing essential CRM features.

## Features
- Customer management
- Add, edit, and delete records
- Search and filter customers
- Responsive user interface
- REST API integration

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## Getting Started

```bash
# Frontend
cd frontend
npm install
npm start

# Backend
cd backend
npm install
npm run dev
```
